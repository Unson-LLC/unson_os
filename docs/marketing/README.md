## 📊 重要なKPI・目標値

### 広告運用の成功指標
- **CTR**: 3%以上
- **CVR**: 10%以上（LP→無料登録）
- **CPA**: 月額料金の50%以下
- **ROAS**: 300%以上（3ヶ月時点）

### プロダクト評価基準
- **7日間評価**: CTR 1%未満 → 広告停止
- **14日間評価**: CVR 5%未満 → LP改善または終了
- **30日間評価**: ROAS 100%未満 → プロダクト見直し
