# UnsonOS UI設計書（完全版）

> 最終更新: 2025-01-15
> バージョン: 2.4.1

## 📌 概要

UnsonOSの管理ダッシュボードは、100-200個のマイクロSaaSプロダクトを最小限のリソースで効率的に管理するためのUIシステムです。

### 設計原則
- **STAY SMALL哲学**: 30分の朝の巡回で全体管理
- **記号化による情報圧縮**: 時系列データを5段階の記号で表現
- **PKGベースの自動化**: インジケーター駆動の条件分岐
- **DAOコミュニティ統合**: 収益の40%自動分配の可視化

## 🚶 7. ユーザーフロー

### 7.1 朝の巡回フロー（30分）

```mermaid
graph TD
    A[ログイン] --> B[システムバージョン確認]
    B --> C[収益・DAO配当確認]
    C --> D[記号マトリックスで異常確認]
    D --> E{異常あり?}
    E -->|Yes| F[時系列データ分析]
    E -->|No| G[サンセット候補確認]
    F --> G
    G --> H{サンセットあり?}
    H -->|Yes| I[延命/終了判断]
    H -->|No| J[Gate承認処理]
    I --> J
    J --> K[新規アイデア確認]
    K --> L[巡回完了]
```

### 7.2 Gate承認フロー（3ステップ）

1. **要件確認** → チェックリスト確認
2. **承認理由** → 判断根拠入力
3. **最終確認** → 実行（取り消し不可）
