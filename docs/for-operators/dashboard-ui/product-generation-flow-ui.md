# UnsonOS プロダクト生成フローUI設計

## 🎯 設計方針

**STAY SMALL哲学**: 最小限のクリックで最大限の価値創造
- 余計なステップを排除
- AI提案を信頼し、人間は重要な判断のみ
- リアルタイムフィードバックで安心感を提供

## 🎯 UIデザインの重要な特徴

### 1. **プログレッシブ・ディスクロージャー**
- 必要な情報のみを段階的に表示
- ユーザーが迷わない明確な次のアクション

### 2. **リアルタイムフィードバック**
- 全プロセスでライブ更新
- 不安を与えないプログレス表示

### 3. **自動化の透明性**
- AIの判断根拠を可視化
- 人間が介入できるポイントを明示

### 4. **モバイルファースト**
- 外出先でも確認・対応可能
- 重要な通知のプッシュ配信

### 5. **データドリブン表示**
- 定量的指標を大きく表示
- 感情的判断を排除する設計

この設計により、UnsonOSユーザーは最小限の作業で最大限の価値創造を実現できます。
