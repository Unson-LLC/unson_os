```

💰 収益関連   📈 成長・増加   📉 減少・低下
```

## 🎯 UI設計の重要ポイント

### 1. **グランス可読性**
- 3秒以内で重要指標を把握可能
- 色とアイコンで瞬時に状況判断

### 2. **アクション指向**
- 各画面に明確な次のアクションを配置
- AI推奨を優先的に表示

### 3. **異常検知優先**
- 問題のあるプロダクトを最優先表示
- 自動対応の透明性確保

### 4. **データストーリーテリング**
- 数値だけでなく背景と示唆を提供
- トレンドと将来予測の統合

### 5. **モバイル・デスクトップ両対応**
- 外出先でも重要情報にアクセス
- デバイス特性に最適化した情報密度

この分析UI により、UnsonOSユーザーは100個のSaaSポートフォリオを効率的に監視・最適化できます。
