## 設定時の注意点

### チャンネル作成手順
1. カテゴリを作成（`═ カテゴリ名 ═`）
2. カテゴリ内にチャンネルを追加
3. チャンネル名は `#｜絵文字｜名前` 形式で統一
4. トピック（チャンネル説明）を必ず設定

### 権限設定の基本
- **お知らせ系**: @Admin のみ投稿可能
- **一般チャンネル**: @everyone 投稿可能
- **DAO関連**: 認証済みメンバーのみ

### ボイスチャンネル
各フェーズで必要に応じて追加：
- 🔊｜雑談ボイス
- 🔊｜開発セッション
- 🔊｜週次スタンドアップ
